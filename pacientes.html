Proyecto: Gestor de Pacientes para Fisioterapeuta Agustín

Este sistema fue desarrollado con el objetivo de brindar una solución digital personalizada para el Lic. Agustín, fisioterapeuta profesional, permitiéndole gestionar de forma eficiente la información clínica de sus pacientes. La plataforma está orientada al uso diario en consultorio y fue diseñada teniendo en cuenta las necesidades reales del trabajo terapéutico.

El sitio cuenta con un panel de búsqueda de pacientes que permite localizar rápidamente a cualquier persona ingresada en la base de datos, así como un formulario detallado para visualizar y actualizar la ficha clínica: edad, altura, peso, historia médica, tratamientos realizados y más. Además, incorpora una funcionalidad clave: la posibilidad de adjuntar archivos clínicos o estudios complementarios, mejorando la organización del material del paciente.

La segunda sección permite agregar nuevos pacientes al sistema de manera sencilla, asegurando que cada ficha esté completa y lista para futuras actualizaciones. Todo esto se integra en una interfaz limpia, responsiva y profesional, acompañada de mensajes motivadores como "Cada pequeño avance cuenta", reforzando el enfoque humano del trabajo terapéutico.

La página fue construida utilizando tecnologías como HTML, CSS, JavaScript, PHP y MySQL, logrando un sistema completamente funcional que puede ser alojado en servidores locales o en la nube. Está pensada para ser usada desde PC, tablet o celular, adaptándose al ritmo del día a día del fisioterapeuta.

Tecnologías clave utilizadas:

  • HTML5 y CSS3 para la estructura y el diseño visual.
  • JavaScript para la interactividad del frontend.
  • PHP para la lógica del backend y la conexión con la base de datos.
  • MySQL para el almacenamiento persistente de los datos de pacientes.

Este proyecto fue desarrollado con dedicación y cariño para el hermano de mi novia, Agustín, como parte de mi crecimiento profesional en el desarrollo web y con la intención de aportar herramientas reales a profesionales que cuidan de la salud y el bienestar de las personas.
